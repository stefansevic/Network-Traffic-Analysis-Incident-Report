# Artifacts and Raw Evidence

This directory contains the raw network traffic captures (PCAP) used for the forensic analysis detailed in this project.

## File Descriptions
* **set2.pcap**: A packet capture file containing HTTP traffic with Basic Authentication. This file was used to identify a Brute Force attack pattern and extract compromised credentials.

## Source
The evidence file is part of the "Packet Sleuth" network forensics labs by Ming Chow (Tufts University).
